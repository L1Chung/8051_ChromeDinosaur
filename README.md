# Chrome Dinosaur
AT89S52 connect with 128x64 LCD (ST7920) and 4x4 Membrane keyboard

P0 -> 4x4 keyboard

P1.0、P1.1、P1.2 -> LCD's RS、RW、E

P2 -> LCD's DB
